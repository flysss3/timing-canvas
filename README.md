# Timing Canvas

A lightweight browser tool for animators to sketch timing references before touching a single keyframe.

## What it does

Timing Canvas lets you record the rhythm of an action by drawing with your mouse or finger — the way animators used to do it with fCheck back in the day.

Hit **Record**, hold to draw, and move your hand the way the action feels. Do a few takes. Play it back. In 60 seconds you have a timing map for your shot.

No 3D. No keyframes. Just timing.

## When to use it

- **Before blocking** — figure out where the beats land before committing to poses
- **Physical actions** — punches, falls, weight shifts, effort
- **Locomotion** — trace the up/down rhythm of a walk or run
- **Dialogue** — draw the rhythm of the performance while listening to the audio
- **Overlap and follow-through** — feel how long the tail of a movement lasts
- **Reference study** — watch your reference video and redraw the timing of what you see

## How to use it

| Action | Control |
|---|---|
| Start / Stop recording | `Space` or Record button |
| Draw | Hold mouse button / finger / pen |
| Play back | `P` or Play button |
| Loop playback | Loop button |
| Save a take | `S` or Save Take button |
| Clear canvas | `C` or Clear button |

Works in the browser on desktop and mobile. On iPhone, add it to your Home Screen from Safari for the best experience.
